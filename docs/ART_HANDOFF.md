# はなそう Map Art — Generation Handoff (for Claude chat / ChatGPT)

Copy-paste kit for generating the 20 location panels. Each prompt below is
**fully self-contained** — paste one prompt per image, in any order. No prior
context needed.

## How the art is used (so you know what NOT to draw)

- Each image is a **background panel** for one study area in a Japanese
  learning app. The app draws its own tappable lesson dots, character sprites,
  and labels ON TOP of your image.
- Therefore: **no text, no letters, no signage writing, no buttons, no dots,
  no characters' speech bubbles, no UI** in the image. Shop signs and labels
  should be blank or scribble-suggested.
- Each panel needs a **winding dirt/stone path from the bottom edge to the top
  edge**, staying within the middle ~60% of the width. The path itself must be
  **simple and low-contrast** (the app's dots and labels sit on it and must
  stay readable). The listed number of "stops" is how many dots the app will
  place — leave that much visual breathing room along the path.
- Output: **portrait, 2:3 aspect ratio** (e.g. 1600×2400). JPG/PNG.

## Master style (already baked into every prompt below)

Kawaii storybook watercolor: soft watercolor fills with warm brown ink
outlines, hand-drawn and cozy, never corporate or flat-vector. Palette: cream
`#fff7ea`, warm pink `#ef6fa7`, gold `#f2be45`, ink brown `#4a3328`, pastel
greens and sky blues. Think a gentle Japanese picture-book town — charming,
warm, lightly whimsical. Consistent warm daylight unless the prompt says
otherwise.

## Delivery checklist (back to the app)

1. Name each file exactly as listed and send them over (or drop into
   `assets/map/`).
2. Consistency matters more than perfection — same palette and outline style
   across panels beats any one gorgeous outlier. Generating 2–3 candidates per
   panel and picking the most consistent works well.
3. Start with panels 1 and 2 — that's where daily practice happens, and the
   first delivered panel is what the positioning system gets built on.
